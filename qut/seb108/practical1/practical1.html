Practical 1: An Introduction to Python#




Welcome to this introductory notebook designed to familiarise newcomers with the essentials of Python. Python is renowned for its readability, simplicity, and the extensive range of libraries it offers, such as NumPy for numerical computations, pandas for data manipulation, and Matplotlib for visualisation. These tools significantly simplify the tasks of data analysis and visualisation. This guide aims to introduce you to the fundamental concepts of Python, showcase these critical libraries, and their practical uses within Google Colab.




Google Colab, or Colaboratory, presented by Google Research, is a cloud-based Jupyter notebook service that allows the execution of Python (and R) code directly in your web browser, with no setup required. It’s a valuable resource for collaborative projects, education, or any data science projects, thanks to its ease of use and no-cost access. Colab integrates seamlessly with Google Drive, making it straightforward to store, access, and share your datasets, notebooks, and other files. For beginners to cloud-based development environments, starting with Google Colab is an excellent introduction to the world of collaborative coding platforms. Utilising a Google account to store and manage your datasets in Google Colab is perhaps the most straightforward method for those new to the platform. If you don’t have a Google account yet, you can sign up for free here.

Below are the steps for setting up a Google Colab environment for working with Python:

  1. Open your web browser and navigate to Google Colab.

  2. Sign in to your Google account.

  3. Click on File in the top left corner, then New notebook.

  4. By default, new notebooks are set up for Python, but you can change this by clicking on Runtime in the top left corner, then Change runtime type.

  5. To ensure everything is working correctly, run the following code in a code cell:
