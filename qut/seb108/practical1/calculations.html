Python Calculations#

Let’s start by exploring what a declaration means. Python functions like an advanced calculator capable of handling complex computations. Many Python programs are essentially a series of basic mathematical operations performed on data. To work with more complex data than what a regular calculator allows, Python provides the ability to create variables. These variables serve as placeholders to store values for later use. Execute the cells below in order:

variable_1 = 2

variable_2 = 3

output_1 = variable_1 + variable_2

What do you think the value of output_1 will be? Clicl on the Show code cell output button to reveal the answer.

print(output_1)
Show code cell output Hide code cell output 
5

I simply saved the numbers 2 and 3 into variables, choosing the basic names ‘variable_1’ and ‘variable_2’, and then calculated their sum. Notice the underscore in the variable names. Spaces_aren’t_allowed_in_variable_names, so underscores are commonly used instead. We’ll discuss variable naming best practices later on.

You might wonder, ‘’Why bother declaring those variables and then adding them, instead of directly calculating:’’

2+3
Show code cell output Hide code cell output 
5

In this instance, you’re spot on. If my goal was simply to find the sum of 2 and 3, I could have directly inputted it. Moreover, should there have been a need to store that result, I could have easily done so:

output_2 = 2+3

And now, I can at any time look at that:

print(output_2)
Show code cell output Hide code cell output 
5

Or, I can use it in further calculations:

output_3 = output_1 + output_2

print(output_3)

10

This is a great opportunity to explore the basic mathematical operations Python offers, two of which we’ve already encountered. The following lists the basic mathematical operations in Python:

addition = 5 + 5
subtraction = 5 - 5
multiplication = 5 * 5
division = 5 / 6
exponentiation = 5**2
modulus = 5 % 3

Let’s print out the results of these operations:

print("Addition: ", addition)
print("Subtraction: ", subtraction)
print("Multiplication: ", multiplication)
print("Division: ", division)
print("Exponentiation: ", exponentiation)
print("Modulus: ", modulus)
Show code cell output Hide code cell output 
Addition:  10
Subtraction:  0
Multiplication:  25
Division:  0.8333333333333334
Exponentiation:  25
Modulus:  2

Division always returns a float, even if the result is a whole number. If you want to return an integer, you can use the // operator:

division_int = 5 // 6

print("Division (int): ", division_int)
Show code cell output Hide code cell output 
Division (int):  0

Another useful operation is the round() function, which rounds a float to the nearest whole number:

round(5/6)

1

I can also specify the number of decimal places to round to:

round(5/6, 2)

0.83

Also, the abs() function returns the absolute value of a number:

abs(-5)
Show code cell output Hide code cell output 
5
